## FORMAT STANDARDS SUPPORTED (2025)

### RFC COMPLIANCE
- RFC 3164 (BSD Syslog Protocol) - Legacy support
- RFC 5424 (The Syslog Protocol) - Current standard
- RFC 3339 (Date and Time on Internet: Timestamps) - Timestamp format

### STRUCTURED FORMATS
- CEF (Common Event Format) - ArcSight standard
- LEEF (Log Event Extended Format) - IBM/QRadar standard
- JSON structured logging
- Key-value pair formats
- CSV (Comma Separated Values)

### TRANSPORT PROTOCOLS
- UDP (Port 514) - Traditional
- TCP (Port 514) - Reliable delivery
- TLS (Port 6514) - Encrypted transport
- RELP (Reliable Event Logging Protocol) - rsyslog extension

## CURRENT TRENDS (2025)

### EMERGING FORMATS
- Cloud-native structured logging (JSON-first)
- OpenTelemetry integration
- Metrics + logs correlation
- Zero-trust network logging

### DEPRECATED/LEGACY
- RFC 3164 format (still supported but discouraged)
- Plain UDP transport (security concerns)
- Non-timestamped log formats
- Proprietary vendor formats without RFC compliance

This comprehensive list represents current syslog sources as of 2025, with emphasis on devices commonly supported by major SIEM platforms and those that primarily rely on syslog for external log delivery (especially communications and radio equipment).
